# Système Temps Réel — Arduino & NilRTOS

## Description
Implémentation d'un système temps réel embarqué sur carte Arduino utilisant le noyau NilRTOS.
Le projet met en œuvre la gestion de tâches périodiques, l'ordonnancement préemptif et la 
synchronisation entre tâches concurrentes.

## Fonctionnalités
- Création et gestion de tâches périodiques avec NilRTOS
- Ordonnancement temps réel et gestion des priorités
- Synchronisation inter-tâches (sémaphores)

## Stack technique
- **Langage :** C / C++
- **Plateforme :** Arduino
- **RTOS :** NilRTOS
- **IDE :** Arduino IDE
